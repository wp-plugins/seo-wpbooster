== Description ==
= EN: =
Attention please! World wide version (wpbooster.com) of this SEO plugin is still under development and will be released within 3 to 6 months. Now, we have only local, polish version, which description you can find below the english one. WPBooster SEO plugin is designed especially for advertisement and SEO linkbuilding for blog posts of users registered in WPBooster system.
Advantages: free links on post with similar content (advanced contextual post matching engine) which will improve positions in search engines, full automatic SEO system, solution type "switch on and forget", full control (you don't have to know SEO, system will automatically set best parameters for your blog, but if you want to do so, there is nice control panel ;)), statistics of distributed advertising links.
= PL: =
Plugin WPBooster służy do reklamy i pozycjonowania wpisów na blogach użytkowników należących do systemu. Działanie wtyczki polega na w pełni automatycznym rozdzielaniu linków reklamowych pomiędzy postami powiązanymi tematycznie. Korzyści: darmowe stałe linki tematyczne poprawiające pozycje w wyszukwiarkach, w pełni automatyczny system, rozwiązanie typu "włącz i zapomnij", pełna kontrola oraz statystyki przydzielania linków reklamowych. Dokładny opis działania wtyczki oraz pomoc online: http://wpbooster.pl

== Installation ==
1. Wgraj katalog wpbooster do katalogu /wp-content/plugins/ na serwerze
2. Aktywuj wtyczkę w menu "plugins/wtyczki" na swojej instalacji wordpressa
3. Zarejestruj konto na stronie http://wpbooster.pl/wp-login.php?action=register
4. Przejdź do "ustawień/settings" i odnajdź "WPBooster", wpisz swoją nazwę użytkownika i hasło otrzymane w wiadomości e-mail a następnie
wybierz kategorię tematyczną swojego bloga i aktywuj wtyczkę
5. Na stronie http://wpbooster.pl/wp-admin/ możesz oglądać przyrost linków reklamowych.
